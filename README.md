# 🎓 Precious – Student Management Dashboard

Precious is a responsive Student Management Dashboard built using **HTML, CSS, and JavaScript**. It provides a clean and interactive interface for managing student records, tracking course progress, and monitoring dashboard statistics in real time.

## ✨ Features

* 📥 Add new students through an interactive enrollment form
* 📊 Live dashboard with student statistics
* 🔍 Search students by name or email
* 📌 Filter students by status (Active/Completed)
* 📈 Automatic progress calculations and metrics
* ⚡ Admin quick action panel
* 🛡️ System health status widgets
* 🎨 Modern responsive UI with glassmorphism design
* 🚀 Dynamic updates using JavaScript (no backend required)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

```text
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 How to Run

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Start managing student records instantly.
